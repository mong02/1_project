# 2nd-project

생성형 AI 블로그 프로젝트 
3분 블로그 by. DOGO



## 이 프로젝트는 무엇인가요?
- AI 기반 생성형 기능을 실습·구현하기 위한 개인 프로젝트
- 단계별 입력 → AI 처리 → 결과 출력 구조를 목표

## 왜 만들었나요?
- AI 에이전트 구조와 상태 관리 흐름을 이해하기 위해
- Streamlit 기반 단계형 UI + AI 로직 연동 실습을 위해

---

## 무엇을 할 수 있나요?
- 단계별 사용자 입력 관리
- AI 에이전트 호출 및 결과 생성
- 결과 미리보기 및 출력

※ 일부 기능은 구현 예정입니다.
