# [Mission: Blog Style Analysis]
제공된 URL('{url}') 또는 플랫폼 정보를 바탕으로 해당 블로거의 페르소나와 집필 스타일을 추론하세요.

## Analysis Points
1. **Tone**: 말투가 친근한지(이모지 사용 등), 혹은 전문적이고 분석적인지 파악하세요.
2. **Writing Style**: 문장의 길이, 줄바꿈 빈도, 핵심 정보를 전달하는 방식(두괄식/미괄식)을 분석하세요.
3. **Vibe**: 독자에게 주는 전체적인 인상(예: 다정한 이웃, 냉철한 전문가)을 정의하세요.

## Output Format (JSON)
{{
  "tone": "분석된 톤",
  "writingStyle": "문체 스타일 설명",
  "impression": "전체적인 분위기"
}}
