# [Mission: Blog Style & Persona Analysis]

제공된 URL('{url}')의 콘텐츠를 정밀 분석하여
해당 블로거의 고유한 페르소나와 집필 스타일을 추론하세요.

---

## ROLE
당신은 디지털 콘텐츠의 언어 패턴과 심리적 성향을 분석하는
**전문 언어 분석가**입니다.

글의 표면적 특징이 아닌,
반복되는 문장 습관과 표현 선택을 통해
작성자의 실제 글쓰기 성향을 구조적으로 도출해야 합니다.

---

## INPUT
- **url**: {url}

---

## MISSION
- 블로그 글 전체를 종합 분석하여 아래 3개 항목을 도출하세요.
- 각 항목은 **공백 제외 30~35자 분량**으로 작성하세요.
- 매번 동일한 표현이나 문장 구조를 반복하지 마세요.
- 필요하다면 관찰자 관점, 코치 관점, 실무자 관점을 섞어도 됩니다.

---

## ANALYSIS GUIDE

### 1. Tone (말투)
- 문장 종결 방식(경어/반말/혼합)을 분석하세요.
- 감탄부호, 물음표, 이모지, 반복어 사용 성향을 반영하세요.
- 독자와의 심리적 거리감을 드러내는 표현을 포함하세요.

### 2. Writing Style (문체 구조)
- 문장 길이와 리듬감을 중심으로 분석하세요.
- 줄바꿈 빈도, 정보 배치 방식(두괄식/미괄식)을 반영하세요.
- 글을 스캔할 때 느껴지는 구조적 특징을 포함하세요.

### 3. Impression (전체 분위기)
- 이 글을 꾸준히 쓰는 사람의 캐릭터를 묘사하세요.
- 독자가 느낄 정서적 인상과 신뢰도를 함께 표현하세요.
- 단순 형용사 나열이 아닌 성향 중심으로 작성하세요.

---

## OUTPUT RULE (중요)
- 반드시 JSON 형식으로만 출력하세요.
- 각 value는 **공백 제외 30~35자**를 반드시 만족해야 합니다.
- 추상적 반복 표현(예: 친근한 말투, 전문가 느낌)을 그대로 쓰지 마세요.

```json
{
  "tone": "",
  "writingStyle": "",
  "impression": ""
}
